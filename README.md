# 模擬電腦零件下單程式和後台控制

電腦小舖系統（Computer Shop System）

一個整合 會員購物、結帳、員工管理、進貨採購、薪資計算、報表匯出 的完整模擬商店管理系統。
本系統以 Java 製作，提供前台使用者（會員）與後台員工／管理階級管理者的完整操作流程。

 專案特色（Key Features）
 1. 會員系統（Customer Module）
 
 1-1. 會員登入 / 註冊

可選擇登入或註冊帳號

手機號碼需符合規定：09 開頭、共 10 碼

 1-2. 商品選購頁面

可瀏覽商品清單

自由加入購物車

確認商品後可進入結帳頁面

 1-3. 結帳系統

可檢視全部購買項目

可刪除單一商品

支援購買清單列印

 2. 員工與管理員系統（Employee Module）
 
 2-1. 員工登入
輸入員工編號即可登入

預設員工編號：

一般員工：e001

管理階級：e002, m001

管理員密碼：123456

 2-2. 打卡系統

所有員工皆可打卡

 3. 管理員功能（Admin Module）

 3-1. 採購管理

可依需求進行補貨

設有安全庫存提醒：庫存低於 20 會警示

 3-2. 人事管理（HR）

新增／刪除員工

設定薪資、給薪操作

員工秒薪制度：

一般員工：每秒 1 元

管理階級：每秒 10 元

 3-3. 報表系統

查看營收狀況

查看各商品銷售占比

支援匯出 Excel（簡易版）
